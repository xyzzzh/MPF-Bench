MPF-Bench

A benchmark family for fine-grained visual reasoning in vision-language models.

Paper Code Dataset

MPF-Bench converts natural images into localized completion tasks with controllable difficulty.

Abstract

Vision-language models perform strongly on many multimodal tasks, but fine-grained visual reasoning remains difficult to evaluate in a controlled and reproducible way. MPF-Bench is a programmatically generated benchmark family for fine-grained visual reasoning. Given a natural image, MPF-Bench partitions it into patches, masks one target patch, and asks the model to identify the candidate patch that correctly fills the missing region.

Because each instance is defined directly by construction, MPF-Bench provides deterministic labels, exact scoring, zero-cost annotation, and controllable difficulty through grid size, candidate count, and mask shape. The same verifiable structure also makes MPF reusable as a self-supervised training signal.

Benchmark Overview

MPF-Bench is designed as a benchmark family rather than a single fixed test set.

What MPF-Bench Measures

  • Fine-grained local reasoning
  • Spatial continuity and appearance compatibility
  • Context-aware part-whole inference
  • Performance under controlled difficulty shifts

Key Properties

  • Programmatic annotation
  • Deterministic scoring
  • Controllable difficulty
  • Scalable generation from natural images

Automatic instance generation, evaluation, and secondary training utility from the same task structure.

Main Results

Zero-shot accuracy decreases as local ambiguity increases, especially when the candidate pool grows.

Key Findings

  • Strong VLMs still struggle on hard MPF settings.
  • Candidate count affects difficulty more strongly than finer grids alone.
  • MPF training produces large in-domain gains.
  • Transfer to external benchmarks is moderate.
